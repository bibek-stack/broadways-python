system like git, mercurial, perforce etc.

# Need of version control system
* to maintain control version
*for team collaboration in a project.

the codes of project should be stored somewhere in the cloud. google drive or any other cloud storage sevices can be used for storing code, but they doesn,t provide git support and there will be more manual work for 
team collaboration, so, we use cloud services which is undestand git instead of google drive. the example of such 
cloud services which understand git are github, gitlab, bitbucket etc.
